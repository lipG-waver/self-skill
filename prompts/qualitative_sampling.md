# 质性采样与人工阅读指南

## 三段采样法

将消息按时间分为早期（前1/3）、中期（中1/3）、近期（后1/3），每段随机抽取 60 条中等长度（20-120字）消息。

### 为什么三段？
- 跨期一致的特征 = 真正的个人风格
- 只在某一期出现的 = 情境性的，需要标注但不写入核心 prompt

## 8 个阅读维度

逐条扫描时，按以下维度记录观察：

1. **句式结构**：长句/短句、复合/简单、反问/设问频率
2. **话题切换**：线性推进 vs 跳跃式，转折词使用
3. **幽默方式**：自嘲 / 讽刺 / 谐音梗 / 冷幽默 / 不幽默
4. **关心方式**：直说 / 绕弯 / 通过行动（科普、帮忙、发链接）
5. **知识引用**：学术 / 历史 / 流行文化 / 商业 / 不引用
6. **独特表达**：专属词汇、比喻、造词、缩写
7. **自我定位**：引导者 / 平辈 / 求助者 / 分析者
8. **冲突风格**：直接对抗 / 冷处理 / 讲道理 / 诉诸情感

## 输出格式

每个维度 1-3 句话总结 + 1-2 条原话示例。标注该特征在三段中的一致性（✅三段一致 / ⚠️仅某段出现）。
