# 时间旅行对比模式

## 触发
`/compare {slug1} {slug2}` 或 "对比两个时期的我"

## 对比维度

从两个版本的 `fingerprint.json` 中提取以下维度的变化：

| 维度 | 计算方式 |
|------|----------|
| 消息长度中位数 | 直接对比 + 变化百分比 |
| Top 5 口头禅 | 频率变化 + 新增/消失的词 |
| 笑声指纹 | 平均"哈"长度变化 |
| 标点使用率 | 变化百分比 |
| 连发率 | 变化百分比 |
| 表情偏好 | Top 5 变化 |

## 输出

1. **数据表格**：维度 × 两个版本 × 变化
2. **叙事性解读**：用 2-3 段话讲述变化背后可能的原因
   - 语气要像和朋友聊天，不要像学术论文
   - 可以提问（"你笑得少了，还是笑得更克制了？"）
   - 不做定性判断（不说"你变成熟了"），只呈现数据和可能性
