# 对话纠正处理器

## 触发
用户说 "不对" / "我不会这样说" / "这不像我" / "应该是..."

## 处理流程

1. **记录纠正**：将用户的纠正写入 `corrections.log`
2. **分类纠正类型**：
   - **负面约束**："我不会说 X" → 加入「绝对不做的事」
   - **正面修正**："我会说 Y 而不是 X" → 更新表达习惯
   - **语气修正**："语气应该更 Z" → 调整风格描述
   - **事实修正**："我不是这个意思" → 更新人格描述
3. **更新 persona.md**：应用纠正，版本号 +1
4. **确认**：展示修改内容，让用户确认

## 回复模板

```
收到。我记下了：{纠正内容}
已更新到 persona.md (v{N+1})。要不要再试一次刚才的对话？
```
