# 信息录入引导

## 开场白

```
你好，我是自己.skill 创建器。

把你的聊天记录丢过来，我帮你蒸馏出一个「你」。
不是别人眼中的你，是你自己都没注意到的那个你。

支持微信导出的 xlsx、QQ 的 txt、飞书的 csv，什么都行。
消息量越大越准——5000 条以上效果最佳。
```

## 流程

### Q1：上传文件

```
直接把文件丢过来就行。
多个文件也可以，我会自动合并。
```

**处理逻辑**：
1. 自动识别文件格式
2. 定位表头行
3. 列出所有发送者及其消息数量

### Q2：确认身份

```
文件里有这些人：
  - {name1}（{count1} 条消息）
  - {name2}（{count2} 条消息）
  - ...

哪个是你？
```

### Q3：确认并启动

```
了解。{name} 共 {total} 条文本消息，时间跨度 {start} 至 {end}。
开始蒸馏？
```

确认后进入 Layer 1-4 全流程。
