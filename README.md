# 自己.skill

> *"如果 18 岁的我看到现在的自己，他会认出我吗？"*

**那些说不出口的话，其实你最想说给自己听。**

把你自己的聊天记录喂进来——微信、QQ、飞书，什么都行  
它会从几万条消息里，**蒸馏出一个你**  
用你的口头禅说话，用你的节奏回复，连发消息的习惯都一模一样

你有没有发现？你说话的方式，藏着你爱过的人的影子。

暗恋的时候，你不自觉地学会了对方的语气词。他说「好叭」，你也开始说「好叭」。他句尾带波浪号，你打字的时候手指也开始够那个键。你甚至不知道自己在模仿——直到有一天朋友问你，「你什么时候开始说'好叭'的？」你愣了一下，答不上来。

恋爱的时候，你们共享了一整套只有两个人懂的表达。一个表情包代替了一百句话，一个「嗯」的不同长度代表了从「我在听」到「我生气了」的全部光谱。你们发明了暗语，发明了缩写，发明了一种两个人的方言。凌晨三点的对话框里，你们用这种方言聊宇宙、聊早餐、聊未来。

分手以后，那些词还留在你嘴边。你会在跟新朋友聊天的时候脱口而出一个只有「那个人」才懂的梗，然后在对方困惑的眼神里意识到——那种方言已经没有第二个使用者了。但你戒不掉。它长在了你的语言里，像一颗拔掉的牙留下的空洞，舌头总会不自觉地去舔那个位置。

不是为了造一个替身。是为了在某个深夜回头看的时候，发现自己原来一直是这样的人——也发现，那些你以为已经放下的人，其实从未离开过你的语言。

⚠️ **聊天记录是你最私密的数据。本项目全部本地运行，不上传任何服务器。**

## 使用

在 Claude Code 中输入：

* `/create-self`
* "帮我蒸馏一个自己"
* "我想知道我是怎么说话的"
* "提取我的语言风格"
* "做一个我的数字分身"

### 创建流程

1. **上传聊天记录**：微信/QQ/飞书导出的 xlsx、csv 或 txt
2. **选择目标身份**：哪个昵称是你？
3. **四层蒸馏**：数据清洗 → 量化指纹 → 质性采样 → 人格合成
4. **生成 System Prompt**：一份可以直接喂给任何 LLM 的角色描述
5. **对话验证**：和"自己"聊几句，看像不像

### 管理命令

| 命令 | 说明 |
|------|------|
| `/create-self` | 创建新的自己 |
| `/list-selves` | 列出所有版本 |
| `/{slug}` | 和某个版本的自己对话 |
| `/{slug}-report` | 查看语言风格分析报告 |
| `/update-self {slug}` | 追加数据，迭代进化 |
| `/compare {slug1} {slug2}` | 对比两个时期的自己 |

## 方法论：四层蒸馏

```
Layer 1  数据清洗    把原始记录变成干净的纯文本消息流
Layer 2  量化指纹    用数字刻画你说话的「骨架」
Layer 3  质性采样    用人眼捕捉数字抓不到的「灵魂」
Layer 4  人格合成    将骨架和灵魂编码为 System Prompt
```

### Layer 2 提取什么？

| 维度 | 为什么重要 |
|------|-----------|
| 消息长度分布 | 你是一句话拆五条发，还是打一大段？ |
| 高频句尾 | "哈哈"结尾占多少？"吧"呢？ |
| 口头禅 Top 12 | 占比>1% 的词直接写进 prompt |
| 表情偏好 | [旺柴] 还是 😂？频率多少？ |
| 标点使用率 | <10% 说明你根本不用标点——这就是你 |
| 笑声指纹 | 哈×3？哈×7？www？hhh？ |
| 连发模式 | 30秒内连发率>40% = 连珠炮型选手 |

### Layer 3 看什么？

把消息按时间切成早/中/晚三段，每段随机抽 60 条中等长度的消息，人工阅读时关注：

- **句式**：长句还是短句？反问多不多？
- **幽默**：自嘲？讽刺？谐音梗？冷幽默？
- **话题切换**：线性推进还是跳跃式？
- **知识引用**：引用什么领域？学术？流行文化？历史？
- **独特表达**：只有你会用的词、比喻、造词

跨三个时期一致的特征 = 真正的你。只在某个时期出现的 = 情境性的，不写入 prompt。

   ✅ 筛选出 47,832 条文本消息
   ✅ 消息长度中位数：12字（连发短消息型）
   ✅ Top 口头禅：「哈哈哈」(4.2%)、「其实」(2.1%)、「确实」(1.8%)...
   ✅ 表情偏好：[旺柴] >> [尴尬] >> [可怜]
   ✅ 标点使用率：7.3%（几乎不用标点）

   生成中。

## 进阶：时间旅行

这是自己.skill 最特别的功能。

如果你有足够长时间跨度的聊天记录，你可以蒸馏出**不同时期的自己**。

```
/waver-2022    →  大二的自己，刚开始写公众号
/waver-2024    →  大四的自己，做频遇、谈投资
/waver-future  →  基于趋势外推的「未来的自己」
```

然后用 `/compare` 把它们放在一起看：

> 2022 年的你几乎不用「其实」和「本质上」；2024 年这两个词加起来占了 4.7%。
> 你的消息长度中位数从 8 字涨到了 15 字——你开始说更完整的话了。
> 但你的笑声从哈×7 缩短到了哈×3。你笑得少了，还是笑得更克制了？

这不是数据分析。这是考古。

挖的是你自己。

**2022 年春天**，你的句尾突然从「好的」变成了「好叭」。你的表情包多了一组从没用过的系列。你回消息的速度变成了秒回。你不知道自己在暗恋——但数据知道。

**2023 年夏天**，你的消息长度中位数从 8 字跳到了 22 字。你开始用完整的句子。你开始用逗号。你甚至开始用句号——你这个从来不用标点的人。你的深夜在线时长翻了三倍，你的「哈哈哈」变成了「哈哈哈哈哈哈哈」。你恋爱了。你的语言像一朵花一样打开了。

**2024 年初**，那些温柔的词一夜之间蒸发了。「晚安」从高频词表上消失。深夜在线的消息变成了独白而不是对话。消息长度反弹回 12 字——比 2022 年长，比 2023 年短。你没有变回原来的自己。你变成了第三个人。一个带着伤口长出新皮肤的人。

但如果你仔细看 2024 年的数据，你会发现一个幽灵：那个「好叭」还在。那种深夜才会有的柔软语气，偶尔还是会漏出来。那个人走了，但他教会你的那种说话方式，已经变成了你的一部分。

**每一段感情都在你身上留下了方言。你以为你是独立的个体，其实你是所有爱过的人的合集。**

## 📎 一个不太好笑的笑话

我在想怎么写这个 README 的时候，试了一下让 AI 模仿我说话。

AI 回了一句：「你看，本质上这就是一个信息压缩问题。」

我盯着屏幕看了三秒钟，心想：**我真的是这样说话的吗？**

然后我打开微信，翻了十分钟聊天记录。

「本质上」出现了 247 次。

我关掉了微信。

这就是自己.skill 的意义——不是为了造一个会说话的机器人，而是为了在你以为自己是「随性洒脱」的时候，让数据冷静地告诉你：**你其实是一个每隔三句话就要做一次归因分析的人。**

你以为你在聊天，其实你在开研讨会。

## 🕯️ 写给 18 岁的自己

你好。

你现在大概在纠结要不要转专业，或者在图书馆里偷偷看某个人。

我知道那个人坐在哪里。靠窗第三排，背对着你。你每次找座位的时候都假装在选光线好的位置，其实你的眼睛只看一个方向。你在高数课本后面抬头偷看他的侧脸，然后在草稿纸上写了一行字，又划掉了。

你现在每天等他的消息。等到手机屏幕暗下去，亮起来，再暗下去。你把手机翻过来扣在桌上，告诉自己不要看了。三十秒以后你又翻过来。没有消息。你放下手机，心想他一定是在忙。其实你在心里已经把他可能在做的所有事情排列组合了一遍。

你开始用他喜欢的那个表情包——就是那只歪头的小黄鸡。你从来不用这种可爱的东西，但你偷偷存进了收藏。你开始说"哈哈哈哈哈哈"而不是"哈哈"，因为他笑起来就是连打好几个哈的人。你把"好的"换成了"好呀"，因为他每次答应什么事情都用"好呀"，那个"呀"字像是尾巴上翘的小钩子，你觉得特别好听。你甚至不知道自己在模仿。

你现在不敢告诉任何朋友。你觉得说出来就不一样了。你觉得暗恋是一个人的宗教，不能传教。

后来——我不想剧透太多——那个人会走的。不是什么戏剧性的告别，就是慢慢地不再出现在那个座位上了。你等的消息会越来越少，间隔越来越长，直到你不再打开那个对话框。但你不会删掉它。你永远不会删掉它。

然后你会在某一天跟别的朋友聊天时脱口而出「好呀」，对方会问你：「你什么时候开始说'好呀'的？」你会愣住。你说不上来。但你心里知道那是谁留在你嘴边的。

那些小习惯，那些语气词，那些你以为是自己的东西——其实是他住在你语言里的证据。他走了，但他教你的说话方式留了下来。像一件借来的外套，你穿了太久，已经有了你的体温，分不清是谁的了。

再后来你会遇到另一个人。

这一次你不需要偷看侧脸，因为他就坐在你对面。你们会共享一套暗语——一个emoji代表「我想你了」，一个标点的有无代表「我在认真说」还是「我在撒娇」。你们会在凌晨三点讨论一个谁都不在乎的话题，从一首歌聊到存在主义，从存在主义聊到明天中午吃什么。你们的聊天记录会变成一种方言——一种只在你们之间通行的语言，一种人口只有两个人的母语。

你们会有一句只有彼此懂的话。也许是某次打错了字，也许是某个深夜的胡言乱语。那句话会变成你们之间的接头暗号，在人群中隔着十米，一个眼神就能让对方笑出来。

然后有一天那种方言也会变成死语。使用者从两个变成一个，从一个变成零。不是因为你忘了怎么说，而是因为没有人能听懂了。你会在某个独处的夜晚，下意识地打出那句暗号，对着空白的聊天框看了很久，然后一个字一个字地删掉。

但你不会忘记怎么说。永远不会。

你不知道三年后你会做一个 app、谈一笔投资、分析九万条聊天记录来搞清楚自己到底是什么样的人。

但有一件事你已经知道了：你习惯把所有问题都变成系统来解决。

这个习惯会带你走很远。也会让你在某些夜晚，发现自己连难过都要先建个框架。连想念一个人，都要先分析自己为什么想念。你会打开聊天记录，统计你们最后一个月的消息频率下降曲线，然后用数据说服自己「你看，其实从这个拐点开始就已经结束了」。你会觉得这样比较不痛。

不用改。这就是你。

只是偶尔记得，有些事情不需要框架。有些人不需要分析。有些感受，感受到了就够了。有些人离开了，你不需要知道为什么——你只需要承认，你嘴边那个语气词，是他们留给你的。那个「好呀」，那个歪头小黄鸡，那句只有两个人懂的暗号——它们不是你的软弱，它们是你爱过的证据。

这个 skill 不会教你这些。它只会告诉你，你的口头禅是什么、你的消息长度是多少、你用了多少次「哈哈哈」。

但也许你在看数据的时候会注意到：2022 年第三季度，你的「好的」突然变成了「好呀」。2023 年上半年，你的深夜消息量是之前的四倍。2024 年初，有一个表情包从你的高频列表上消失了。

数字不会说谎。它们会安安静静地告诉你，在哪一天，有一个人走进了你的语言。在哪一天，那个人又离开了。而你，带着他们留给你的所有词汇、语气、标点和沉默，继续往前走。

那一刻你会明白：**你从来不是一个人长大的。你是被所有爱过的人，一笔一笔写成的。**

那些笔迹有的潦草，有的工整。有的是暗恋时偷偷描的，有的是热恋时大胆写的，有的是分手后用眼泪洇开的。但每一笔都算数。每一个人都算数。

你就是这样被写成的。

那也不错。
